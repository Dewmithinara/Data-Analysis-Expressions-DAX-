## üîπ What is DAX?

**DAX (Data Analysis Expressions)** is a formula language used in **Power BI** to perform calculations such as totals, averages, and time-based analysis.

---

## 1Ô∏è‚É£ Create a Table in DAX

Creating a table in DAX means generating a new table inside Power BI using data or logic. These tables are used to organize data and support calculations.

---

## 2Ô∏è‚É£ Add a New Column in DAX

Adding a new column means creating a calculated column that works **row by row**. Each row gets its own calculated value based on other columns in the same row.

---

## 3Ô∏è‚É£ Create Measures in DAX

Measures are calculations that return results based on **filters, slicers, and visuals**. Measures are commonly used for totals, averages, and KPIs.

---

## 4Ô∏è‚É£ DAX Functions (Basic)

Basic DAX functions are used to:

* Add values
* Find averages
* Count rows
* Apply simple conditions

These functions help summarize data easily.

---

## 5Ô∏è‚É£ DAX Functions 2 (Advanced)

Advanced DAX functions are used to:

* Change filter conditions
* Apply calculations on specific data
* Ignore or modify existing filters

These functions help create powerful and flexible calculations.

---

## 6Ô∏è‚É£ DAX Time Intelligence Functions

Time intelligence functions are used to analyze data over time such as days, months, quarters, and years. They require a proper **Date Table**.

## 7Ô∏è‚É£ MTD, QTD, YTD (Simple Meaning)

### üîπ MTD (Month to Date)

Shows results from the **start of the current month until today**.

### üîπ QTD (Quarter to Date)

Shows results from the **start of the current quarter until today**.

### üîπ YTD (Year to Date)

Shows results from the **start of the current year until today**.
